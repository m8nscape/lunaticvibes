SDL_assert_state

An enumeration of assertion handling states.

Values

SDL_ASSERTION_RETRYretry the assert immediately
SDL_ASSERTION_BREAKtrigger a breakpoint under the debugger
SDL_ASSERTION_ABORTterminate the program
SDL_ASSERTION_IGNOREignore the assert
SDL_ASSERTION_ALWAYS_IGNOREignore the assert from now on

Code Examples

Remarks

This enumeration is returned by the callback function in SDL_SetAssertionHandler() to determine the response to failed assertions.
