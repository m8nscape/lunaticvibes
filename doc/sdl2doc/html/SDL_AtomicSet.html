SDL_AtomicSet

Use this function to set an atomic variable to a value.

Syntax

int SDL_AtomicSet(SDL_atomic_t* a,
                  int           v)

Function Parameters

aa pointer to an SDL_atomic_t variable to be modified
vthe desired value

Return Value

Returns the previous value of the atomic variable.
