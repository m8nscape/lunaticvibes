SDL_CalculateGammaRamp

Use this function to calculate a 256 entry gamma ramp for a gamma value.

Syntax

void SDL_CalculateGammaRamp(float   gamma,
                            Uint16* ramp)

Function Parameters

gammaa gamma value where 0.0 is black and 1.0 is identity
rampan array of 256 values filled in with the gamma ramp
