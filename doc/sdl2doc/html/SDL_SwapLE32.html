SDL_SwapLE32

Use this function to swap the byte order of a 32-bit little-endian value to native ordering.

Syntax

Uint32 SDL_SwapLE32(Uint32 x)

Function Parameters

xthe value to be swapped

Return Value

Returns the native 32-bit value.
