SDL_RenderFillRect

Use this function to fill a rectangle on the current rendering target with the drawing color.

Syntax

int SDL_RenderFillRect(SDL_Renderer*   renderer,
                       const SDL_Rect* rect)

Function Parameters

rendererthe rendering context
rectthe SDL_Rect structure representing the rectangle to fill, or NULL for the entire rendering target

Return Value

Returns 0 on success or a negative error code on failure; call SDL_GetError() for more information.
