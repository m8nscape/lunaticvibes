SDL_ConvertPixels

Use this function to copy a block of pixels of one format to another format.

Syntax

int SDL_ConvertPixels(int         width,
                      int         height,
                      Uint32      src_format,
                      const void* src,
                      int         src_pitch,
                      Uint32      dst_format,
                      void*       dst,
                      int         dst_pitch)

Function Parameters

widththe width of the block to copy, in pixels
heightthe height of the block to copy, in pixels
src_formatthe format of the source pixels; see Remarks for details
srca pointer to the source pixels
src_pitchthe pitch of the block to copy
dst_formatthe format of the destination pixels; see Remarks for details
dsta pointer to be filled in with new pixel data
dst_pitchthe pitch of the destination pixels

Return Value

Returns 0 on success or a negative error code on failure; call SDL_GetError() for more information.
