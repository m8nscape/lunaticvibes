SDL_SetRenderTarget

Use this function to set a texture as the current rendering target.

Syntax

int SDL_SetRenderTarget(SDL_Renderer *renderer,
                        SDL_Texture  *texture)

Function Parameters

rendererthe rendering context
texturethe targeted texture, which must be created with the SDL_TEXTUREACCESS_TARGET flag, or NULL for the default render target

Return Value

Returns 0 on success or a negative error code on failure; call SDL_GetError() for more information.

Code Examples

Remarks

Before using this function, you should check the SDL_RENDERER_TARGETTEXTURE bit in the flags of SDL_RendererInfo to see if render targets are supported.
