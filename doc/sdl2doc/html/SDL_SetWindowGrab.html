SDL_SetWindowGrab

Use this function to set a window's input grab mode.

Syntax

void SDL_SetWindowGrab(SDL_Window* window,
                       SDL_bool    grabbed)

Function Parameters

windowthe window to change
grabbedSDL_TRUE to grab input, SDL_FALSE to release input

Code Examples

Remarks

When input is grabbed the mouse is confined to the window.
