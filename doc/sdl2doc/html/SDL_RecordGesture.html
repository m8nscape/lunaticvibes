SDL_RecordGesture

Use this function to begin recording a gesture on a specified touch device or all touch devices.

Syntax

int SDL_RecordGesture(SDL_TouchID touchId)

Function Parameters

touchIdthe touch device index, or -1 for all touch devices

Return Value

Returns 1 on success or 0 if the specified device could not be found.

Code Examples

Remarks

If the parameter is -1 (i.e., all devices), this function will always return 1, regardless of whether there actually are any devices.
