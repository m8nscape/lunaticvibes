SDL_GetWindowSize

Use this function to get the size of a window's client area.

Syntax

void SDL_GetWindowSize(SDL_Window* window,
                       int*        w,
                       int*        h)

Function Parameters

windowthe window to query
wa pointer filled in with the width of the window
ha pointer filled in with the height of the window
