SDL_JoystickIsHaptic

Use this function to check to see if a joystick has haptic features.

Syntax

int SDL_JoystickIsHaptic(SDL_Joystick* joystick)

Function Parameters

joystickthe SDL_Joystick to test for haptic capabilities

Return Value

Returns 1 if the joystick is haptic, 0 if it isn't, or a negative error code on failure; call SDL_GetError() for more information.
