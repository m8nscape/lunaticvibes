SDL_SwapLE64

Use this function to swap the byte order of a 64-bit little-endian value to native ordering.

Syntax

Uint64 SDL_SwapLE64(Uint64 x)

Function Parameters

xthe value to be swapped

Return Value

Returns the native 64-bit value.
