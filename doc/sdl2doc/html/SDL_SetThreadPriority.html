SDL_SetThreadPriority

Use this function to set the priority for the current thread.

Syntax

int SDL_SetThreadPriority(SDL_ThreadPriority priority)

Function Parameters

prioritythe SDL_ThreadPriority to set; see Remarks for details

Return Value

Returns 0 on success or a negative error code on failure; call SDL_GetError() for more information.

Code Examples

Remarks

priority may be one of the following:
SDL_THREAD_PRIORITY_LOWfor non-urgent, background processing
SDL_THREAD_PRIORITY_NORMALfor general purpose processing (default)
SDL_THREAD_PRIORITY_HIGHfor timing-critical processing
