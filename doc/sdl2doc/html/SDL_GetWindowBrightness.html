SDL_GetWindowBrightness

Use this function to get the brightness (gamma correction) for a window.

Syntax

float SDL_GetWindowBrightness(SDL_Window* window)

Function Parameters

windowthe window to query

Return Value

Returns the brightness for the window where 0.0 is completely dark and 1.0 is normal brightness.
