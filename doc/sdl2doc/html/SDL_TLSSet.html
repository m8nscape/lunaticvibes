SDL_TLSSet

Use this function to set the value associated with a thread local storage ID for the current thread.

Syntax

int SDL_TLSSet(SDL_TLSID   id,
               const void* value,
               void (*destructor)(void*))

Function Parameters

idthe thread local storage ID
valuethe value to associate with the ID for the current thread
destructora function called when the thread exits, to free the value

Return Value

Returns 0 on success or a negative error code on failure; call SDL_GetError() for more information.

Code Examples

Remarks

The function prototype for destructor is: 

  •     void destructor(void* value)
  • where its parameters are
  • valuewhat was passed as value to SDL_TLSSet()

Version

This function is available since SDL 2.0.0.
