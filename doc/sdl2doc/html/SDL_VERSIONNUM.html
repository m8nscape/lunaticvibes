SDL_VERSIONNUM

Use this macro to convert separate version components into a single numeric value.

Syntax

SDL_VERSIONNUM(X, Y, Z)

Function Parameters

Xmajor version; reported in thousands place
Yminor version; reported in hundreds place
Zupdate version (patchlevel); reported in tens and ones places

Code Examples

Remarks

This assumes that there will never be more than 100 patchlevels.

  • Example:SDL_VERSIONNUM(1,2,3) -> (1203)
