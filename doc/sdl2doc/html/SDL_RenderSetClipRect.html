SDL_RenderSetClipRect

Use this function to set the clip rectangle for rendering on the specified target.

Syntax

int SDL_RenderSetClipRect(SDL_Renderer*      renderer,
                          const SDL_Rect*    rect)

Function Parameters

rendererthe rendering context
rectan SDL_Rect structure representing the clip area, or NULL to disable clipping

Return Value

Returns 0 on success or a negative error code on failure; call SDL_GetError() for more information.
