SDL_SwapLE16

Use this function to swap the byte order of a 16-bit little-endian value to native ordering.

Syntax

Uint16 SwapLE16(Uint16 x)

Function Parameters

xthe value to be swapped

Return Value

Returns the native 16-bit value.
