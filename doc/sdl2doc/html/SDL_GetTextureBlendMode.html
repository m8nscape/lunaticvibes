SDL_GetTextureBlendMode

Use this function to get the blend mode used for texture copy operations.

Syntax

int SDL_GetTextureBlendMode(SDL_Texture*   texture,
                            SDL_BlendMode* blendMode)

Function Parameters

texturethe texture to query
blendModea pointer filled in with the current SDL_BlendMode; see Remarks for details

Return Value

Returns 0 on success or a negative error code on failure; call SDL_GetError() for more information.

Code Examples

Remarks

blendMode will be filled in with one of the following:
SDL_BLENDMODE_NONEno blending
 dstRGBA = srcRGBA
SDL_BLENDMODE_BLENDalpha blending
 dstRGB = (srcRGB * srcA) + (dstRGB * (1-srcA))
 dstA = srcA + (dstA * (1-srcA))
SDL_BLENDMODE_ADDadditive blending
 dstRGB = (srcRGB * srcA) + dstRGB
 dstA = dstA
SDL_BLENDMODE_MODcolor modulate
 dstRGB = srcRGB * dstRGB
 dstA = dstA
