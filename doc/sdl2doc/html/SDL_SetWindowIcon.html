SDL_SetWindowIcon

Use this function to set the icon for a window.

Syntax

void SDL_SetWindowIcon(SDL_Window*  window,
                       SDL_Surface* icon)

Function Parameters

windowthe window to change
iconan SDL_Surface structure containing the icon for the window
